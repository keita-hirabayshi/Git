gitの使用する目的 ・ファイルのバージョンを管理するため（複数人で編集していると、誰が何を変更したか把握しにくいことがある） → 誰がいつ何を編集したのか分かりやすくなる。 Git リーマス・トーマルズ Linux開発で使用していたバージョン管理システムのライセンスが変更され使用できなくなり、独自の管理システムを開発を提案(Git) グループでの開発に適していなかったから、 GitHub(Gitで作成したコードをオンライン上で扱ってくれるサービスのこと) クリス・ワンストラス 友達とコードをシェアするGitレポジトリホスティングサービスが欲しくて作られた。(公開したいならGitHub、非公開にしたいならBitbucket) GitHubの利点 ・プルリクエストで複数人開発可能、世界中のチームがGitHubで開発(積極的にコード開発に参加できるsocial codingの場になっている)
